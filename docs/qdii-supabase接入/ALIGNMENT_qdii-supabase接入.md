# ALIGNMENT — QDII 基金罗盘 · Supabase 接入

## 原始需求
做一个 QDII 基金大全的 Web App，含所有 QDII 基金、简介、价值等信息，方便查询和对比。

## 现状（项目分析）
已有原生 JS 项目位于 `/Users/rs/Desktop/project/基金分析`：

- **后端**：`server.mjs`（Node http 服务），端口 5173
- **前端**：`public/index.html` + `public/app.js` + `public/styles.css`
- **数据源**：东方财富基金排行 + 基金代码库 + F10 详情页
- **缓存**：本地 `.cache/qdii-funds.json`，TTL 30 分钟
- **已有功能**：基金列表、搜索筛选、对比（最多 6 只）、详情抽屉、配置观察分

## 验收标准
1. 项目 `npm start` 能跑起来，访问 `http://localhost:5173` 可看到完整列表
2. 列表数据来自 Supabase（断网时本地无缓存也能用旧数据展示）
3. 详情页能展示该基金的净值走势图（至少展示已采集到的历史点）
4. 用户可以注册/登录，登录后可收藏基金，未登录看不到收藏入口
5. 重启服务，数据不丢
6. `.env` 文件不进 git
