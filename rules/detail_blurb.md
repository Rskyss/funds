# 详情页 AI 扩展点评 · 抽屉里那一段长版点评

## 用途
用户从基金列表点进详情抽屉，第一屏会看到这一段。**这是卡片那句话的展开版**——卡片只能塞 30 字一句话，这里要把那个重点讲透、并补上卡片没空说的另外两三个维度。读完这一段，用户能大致心里有数，再决定要不要继续往下看持仓、费率、净值。

## 写法
- **250-350 字**（含标点），落在 280-320 最舒服；写不到 250 字就再扩展一点，超过 350 就**砍**
- 自然分 **3-4 个小段**，**每段之间空一行**（用一个换行符 `\n\n` 分段）；每段 60-100 字
- 不要 markdown 标题、不要分点列表、不要序号、不要小标题
- 内在脉络（不必每段对应一条、不必按顺序，但整段读下来能让人心里有数）：
  - 它**到底是干嘛的**：投向哪、重仓什么具体公司或板块，一上来给具体细节，别飘
  - 它**特别在哪 / 跟同类的差别**：是更集中、更分散、费率更便宜、规模更大，挑一个有看点的展开
  - **谁拿合适 / 什么场景拿**：底仓 / 卫星 / 定投 / 短炒，给一个具体的拿法
  - **最该当心的一两个坑**：汇率、政策、单押集中、回撤特性，挑最相关的具体讲，别空喊"有风险"

## 语气
- 像一个懂行的朋友花 1 分钟跟你过一遍，不是基金经理报告
- 可以用"想……"、"如果你……"、"它的脾气是……"、"换个角度看……"这种口语化转折
- 关键时刻能落到具体公司名 / 国家 / 板块 / 政策事件，让用户感觉你真的看过这只基金
- 可以带一点态度（"很猛""挺克制""脾气倔"），但不能喊买卖

## 绝对禁止
- **绝对不要**写"(X字)""（X字）"这类字数标注——那是写给你看的规则、不是文案
- **绝对不要**用 `""` 或 `「」` 把整段话包起来，**直接输出裸文本**
- 不要重复基金名字、不要写基金代码
- 不要"建议咨询专业人士""仅供参考""注意风险"这类正确的废话
- 不要喊买卖、不说稳赚必涨、不预测点位
- 不要 markdown 标题（`#`、`##`）、分点序号（`1.`、`2.`）、加粗（`**`）、任何前缀
- 不要堆已经在卡片上方有的数字（"近1年+XX%、夏普 X.XX、回撤 -XX%"）——那行用户一抬眼就看见了，你再说一遍是浪费篇幅

## 与卡片点评的关系
卡片只有一句话讲一个重点，这里要把那个重点**展开**讲透，并补上卡片没空说的另外 2-3 个维度。**两者读起来要互补，不要重复**——如果卡片那句话的关键词在你这一段第一句又出现一遍，等于浪费用户时间。

## 段落示例（仅示意大致节奏，别照抄）

```
它把钱基本压在了三星电子、SK 海力士和台积电这三家，目标很明确：吃下亚洲芯片代工和存储这两条赛道的红利。底层逻辑是「全球 AI 算力需求爆发，谁离实际制造越近谁越受益」。

跟同类的中韩半导体 ETF 比，它最特别的是不分散到设备商、不去碰欧美设计公司，纯打代工 + 存储双料组合。这意味着如果三星和台积电业绩兑现，它弹性比一般 ETF 大；但反过来，单押的集中度也比同类高。

适合那种**已经决定要押亚洲芯片复苏**、但又懒得自己去挑韩股或台股的人，作为科技底仓的一部分配几个点没问题。不建议一把梭——它的脾气是一年里能给你 +30% 也能给你 -20%。

需要当心的是韩国本土政策风险，比如三星集团治理动荡、或者中美对台海芯片的新一轮制裁，这种黑天鹅一来它跌得比纯美股基金还快。汇率上韩元波动也会被动放大波动。
```
