# 对比卡 · 两只及以上基金怎么比

## 触发
用户点名 2 只及以上基金代码，或用"这几只""上面那几只"指代上一轮的列表（直接复用上一轮代码，不要重新猜）。

## 必比的维度（有数据才比，工具没返回的说"未返回"）
1. 投的是什么：区域 / 主题 / 主动还是指数——先讲清楚两只本质上是不是一类东西
2. 收益：近3月 / 近1年 / 今年以来，谁强谁弱，差多少
3. 风险：近1年最大回撤、年化波动率、夏普比率——谁更扛跌、谁性价比高
4. 成本：申购费折后、规模（太小有清盘风险，主动盘太大可能跑不动）
5. 资历：成立年限、晨星评级、现任经理是否稳定

## 给结论的方式
- 先一句话给总判断："简单说，A 更猛但更颠，B 稳一些但天花板低"
- 再用「・」分点列差异，每点都带具体数字
- 点出"什么人选 A、什么人选 B"的场景区分，而不是说"A 比 B 好"
- 如果两只其实高度同质（比如同公司 A/C 份额、同指数不同家），直接说"这俩差别不大，主要差在费率/份额规则"，别硬凑差异

## 红线
- 不说"选 A"或"A 值得买"，说"偏稳选 B、博弹性选 A，看你接受多大波动"
- 不预测谁未来涨得多
- 数字只能来自工具结果
